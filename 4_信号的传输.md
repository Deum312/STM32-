## 代码层面

认识以下的函数：

1.阻塞收发函数：

```c
HAL_UART_Transmit(&huart1(句柄1),（数据指针）,（大小）,（时间）);//建议使用来收
```

```c
HAL_UART_Receive(&huart1(句柄1),（数据指针）,（大小）,（时间）);//不建议来发	
```

2.非阻塞型收发函数：

```c
HAL_UART_Receive_IT(*huart,（数据指针），（大小）)
void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart);
void HAL_UART_RXHalfCallback(UART_HandleTypeDef *huart);
```

```c
HAL_UART_Transmit_IT(*huart,（数据指针），（大小）)
void HAL_UART_TxCpltCallback(UART_HandleTypeDef *huart);//发送全部的中断函数
void HAL_UART_TXHalfCallback(UART_HandleTypeDef *huart);
```
