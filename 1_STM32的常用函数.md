# 1.stm32的cube的配置

1. 查找芯片(STMF103RB)

2. 选择**Pinout & configuration**中的system core
   1. 先进行RCC的配置high speed clock/low speed clock为crystal
   2. 网上查找你当前的芯片的最大时钟（因为要求最佳性能）,然后图片中（72M是这个芯片的最大晶振）
      然后按照下面配置**Clock Configuration**(主要的目的是为了让总的晶振为72M)：
      ![RCC外部晶振配图](C:\Users\CJX\Desktop\笔记\STM32\图片\RCC外部晶振配图.png)
   3. sys调试接口Debug设置为：Serial Wire
   4. 然后在图上进行定义引脚。(注意每个引脚有八种模式，要选择对应的模式填写)
   5. GPIO点开进行以下的配置：（英文自查）
      ![image-20210619110659714](C:\Users\CJX\Desktop\笔记\STM32\图片\GPIO的配置.png)
   
3. Project Manager中的code Generator
   第一和二个选择第一个
   
4. project中的toollchain写成你相对应的编辑器(这里写V5)

   # 代码层面

   1. 要自己学会去gpio.h里面去找，或者是上网查，就那几个函数而已
   2. 软件具有补全功能，可以不用去记它，但是每个函数的括号里面的参数要花费时间去记住他。
